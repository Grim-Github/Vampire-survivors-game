# Quin Survivors (Vampire Survivors-like Fan Game)

A fast, horde-survival roguelite where **chatters** swarm the arena and you (as Quin) try to outscale the clown fiesta.  
Auto-attacks, passives, and chat-driven chaos. 
> ⚠️ **Fan project** — not affiliated with Vampire Survivors or Quin69. Non-commercial, for fun and learning.
